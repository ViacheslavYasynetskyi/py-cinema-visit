Відвідування кінотеатру
Прочитайте інструкцію перед початком
Ви відкрили власний кінотеатр. Щоб мати краще уявлення про те,
 що відбувається в кінотеатрі, ви вирішили вести записи про події
 в кінотеатрі. Для цього необхідно створити такі модулі:

У каталозі appстворити пакет cinema. У цьому пакеті створіть модулі:

bar.py- всередині цього модуля створити CinemaBar клас,
 який описує роботу cinema bar. Цей клас повинен мати лише
 один статичний метод sell_product, який приймає product- назву
 продукту, який хоче клієнт, і customer- Customerекземпляр,
 що означає клієнта. Цей метод друкує, який продукт і кому продав
 кінотеатр.
cb = CinemaBar()
customer = Customer("Bob", "popcorn")
cb.sell_product(customer=customer, product=customer.food)
# Cinema bar sold popcorn to Bob.
hall.py- всередині цього модуля створити CinemaHall клас,
 який описує дії під час сеансу фільму. Його конструктор бере і
 зберігає number- номер залу в кінотеатрі. Цей клас повинен мати
 лише один метод movie_session, який приймає
 movie_name: customers- список клієнтів ( Customerекземпляри),
  cleaning_staff- очищувач ( Cleaner екземпляр).
  Цей метод друкує про початок фільму, викликає метод
  клієнтів watch_movie, друкує про кінець фільму,
  викликає метод cleaner clean_hall.
У каталозі appстворити пакет people. У цьому пакеті створіть модулі:

customer.py- всередині цього модуля створюється Customerклас,
його конструктор бере і зберігає name, food- їжа,
 яку клієнт хоче купити в барі кінотеатру. Цей клас повинен мати
 лише один метод watch_movie, цей метод приймає movieта друкує,
  який фільм дивиться клієнт.
bob = Customer(name="Bob", food="popcorn")
bob.watch_movie(movie="Madagascar")
# Bob is watching "Madagascar".
cinema_staff- всередині цього модуля створюється Cleanerклас,
його конструктор приймає та зберігає name. Цей клас повинен мати
лише один метод clean_hall, цей метод приймає hall_numberномер залу,
 який прибиральник має прибрати, і друкує, що прибиральник прибирає
 цей зал.
anna = Cleaner(name="Anna")
anna.clean_hall(hall_number=5)
# Cleaner Anna is cleaning hall number 5.
У модуль main.pyви повинні імпортувати всі ці класи.
 Класи слід імпортувати за абсолютним шляхом,
  який починається з "app". з ключовим словом "від".
  Напишіть функцію cinema_visit, яка приймає movie_name,
  customers- список клієнтів, елементи - це диктовки з 'ім'ям' і
   бажаною 'їжею' клієнта, cleaning_staff- ім'я прибиральника,
    який буде прибирати зал після кіносеансу.

Ця функція має створити Customersекземпляри,
екземпляр CinemaHall і CinemaBar, екземпляр Cleaner.
 Спочатку кінобар повинен продавати їжу клієнтам,
 потім кінозал повинен зробити сеанс кінотеатру і, нарешті,
  прибиральник прибирає кінозал.

приклад:

customers = [
    {"name": "Bob", "food": "Coca-cola"},
    {"name": "Alex", "food": "popcorn"}
]
hall_number = 5
cleaner_name = "Anna"
movie = "Madagascar"
cinema_visit(customers=customers, hall_number=5, cleaner="Anna", movie="Madagascar")
# Cinema bar sold Coca-cola to Bob.
# Cinema bar sold popcorn to Alex.
# "Madagascar" started in hall number 5.
# Bob is watching "Madagascar".
# Alex is watching "Madagascar".
# "Madagascar" ended.
# Cleaner Anna is cleaning hall number 5.
